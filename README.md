
## Usage
```
node . [options]
```

## Options
```
--test
  test connection

--init
  initialize remote machine

--deploy
    deploy config

--jump
    use jump host

-d, --domain 
    specify domain

-e, --env
    specify environment

-m, --message
    commit message
```


## Example
```
node . --jump --deploy -d demo -e uat
```
